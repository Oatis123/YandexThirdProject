Техническое задание: Telegram-бот "ProgHelper AI"

Цель проекта
Создать Telegram-бота для помощи в изучении программирования, который:
- Объясняет базовые темы и алгоритмы
- Отвечает на вопросы с помощью AI
- Позволяет просто пообщаться с AI на тему программирования (и около неё)

Функционал
1. Каталог тем
   Пользователь может выбрать из списка тем:
   - Основы Python (переменные, условия, циклы, функции, классы)
   - Основы алгоритмов (сортировки, поиск, рекурсия, жадные алгоритмы и т.д.)
   - Основы структуры данных (стек, очередь, дерево, граф)
   
2. Объяснение тем
   При выборе темы бот высылает короткое объяснение + пример кода.
   Можно запросить "Подробнее" — бот отправит более глубокое объяснение.

3. Режим "Помощь от AI"
   Пользователь пишет свой вопрос (например, "Объясни быструю сортировку"), бот через нейросеть (GPT, локально или через API) отвечает.

4. Чат-режим с AI
   Пользователь включает режим общения, бот становится "другом-программистом": можно спрашивать что угодно по теме кода, обучения или даже попросить совет.

5. Меню команд
   - /start — Приветствие и краткое объяснение возможностей
   - /topics — Список доступных тем
   - /ask — Задать вопрос AI
   - /chat — Включить свободный чат с AI
   - /help — Памятка по командам

Требования к реализации
- Python 3.11+
- Библиотека aiogram для управления ботом
- Подключение к OpenAI API (или локальный LLM по желанию)
- Структурированный код: разнести команды, обработчики сообщений и генерацию ответов по файлам
- Минимальное логирование для отладки ошибок
